void *Map(VkDeviceSize offset = 0, VkDeviceSize size = VK_WHOLE_SIZE)¶

Maps this ManagedDeviceMemory at a given offset with a given size. NOTE : If this ManagedDeviceMemory is already mapped with a different offset or size than the ones provided it will be unmapped and remapped with the provided offset and size. This will invalidate any outstanding pointers into this ManagedDeviceMemory’s mapped memory.

Parameters

  • offset: The offset in bytes to map this ManagedDeviceMemory at (optional = 0)

  • size: Ths size in bytes to map (optional = VK_WHOLE_SIZE)
