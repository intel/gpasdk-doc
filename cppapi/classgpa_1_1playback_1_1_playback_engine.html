void DisableCalls(uint64_t const *callIndices, uint64_t count)¶

Skips execution of the API calls at the given indices.

This additively changes the state; any unique calls that were specified as disabled previously will remain disabled. This doesn’t overwrite previous calls to

DisableCalls(); use EnableCalls() to re-enable API calls. This disables only the execution of the given call, and does not affect pre- or post- call notifications that CallableReceivers may get. Note that disabling some graphics calls may prove fatal to the execution of playback.
Parameters

  • callIndices: A list that contains the indices of the API calls that should not be executed. If null, this routine returns early. count The number of elements in callIndices.

void EnableCalls(uint64_t const *callIndices, uint64_t count)¶

Re-enables execution of calls previously disabled by DisableCalls().

Parameters
