void GetResourceInfoExs(ResourceInfoUsage usage, uint64_t callIndex, size_t *pResourceInfoExCount, ResourceInfoEx const **ppResourceInfoExs) const¶

Gets all registered ResourceInfoExs associated with this ResourceInfo for a specified ResourceInfoUsage at a specified call index.

Parameters

  • [in] usage: The ResourceInfoUsage to get the registered ResourceInfoExs for

  • [in] callIndex: The call index to get the registered ResourceInfoExs for

  • [inout] pResourceInfoExCount: A pointer to the number of registered ResourceInfoExs

    NOTE : If ppResourceInfoExs is nullptr this argument will be populated with the total number of registered ResourceInfoExs

    NOTE : If ppResourceInfoExs is not nullptr this argument will be used to determine the number of

    ResourceInfoEx pointers to write to ppResourceInfoExs

  • [out] ppResourceInfoExs: A pointer to an array of ResourceInfoEx pointers to populate
