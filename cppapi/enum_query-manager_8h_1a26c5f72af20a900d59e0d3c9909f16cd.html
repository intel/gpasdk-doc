Enum Documentation¶

enum gpa::metrics::QueryManagerType¶

QueryManager object types (that implement the IQueryManager interface) and are used to accurately query metrics for a graphics API call or range of calls.

Values:

kInvalid¶

Invalid IQueryManager implementation.

kUnspecified¶

IQueryManager implemention is determined based on platform specific runtime logic.

kMetalStatistics¶

Apple Metal Statistics API IQueryManager implementation (available only on macOS for Metal).

kMetricsDiscoveryApi¶

Intel MD performance query-based IQueryManager implementation (available on Linux for Vulkan, macOS for Metal, and Windows for both D3D12 and Vulkan).

kVulkanStatisticsAndTimeStamp¶

Vulkan API pipeline statistics & timestamp query based IQueryManager implementation (available only on Linux and Windows for Vulkan).

kDefault = QueryManagerType::kUnspecified
