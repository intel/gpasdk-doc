template<typename EndPointType, typename ValueType>
class gpa::playback::detail::IntervalTree¶

A sorted associative container of Intervals<> that provides enumeration over all Intervals<> that overlap a given interval.

Parameters

  • <ValueType>: The type of value to associate with each Interval<>
