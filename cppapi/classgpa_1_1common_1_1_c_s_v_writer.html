void Flush()¶

Flushes all previously written values to the current file handle.

void Close()¶

Closes the file handle.

Valid() will return false from this point and no more data can be written. Closing will also perform as part of the destructor; this method facilitates user control over when closing happens.

bool Valid() const¶

Returns true if the current file handle is valid.
