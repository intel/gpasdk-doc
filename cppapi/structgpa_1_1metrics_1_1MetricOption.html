struct MetricOption

Structure that contains information about an option for a prototype metric or instance of one.

Public Types

enum class Type

Types of prototype or instantiated metric options.

Values:

enumerator kDisaggregation

Option type used for enabling disaggregation.

enumerator kLatency

Option type used for enabling latency.

enumerator kNormalizationAverage

Option type used for enabling average normalization (raw metric / instance count). Normalization types are mutually exclusive.

enumerator kNormalizationRate

Option type used for enabling rate normalization (raw metric / timestamp). Changes metric units to per second. Normalization types are mutually exclusive.

enumerator kNormalizationUtilization

Option type used for enabling utilization normalization (raw metric * 100% / cycles / instance count). Changes metric units to percent. Normalization types are mutually exclusive.

enumerator kCount

Number of possible option types.

enumerator kInvalid

Invalid option type.
