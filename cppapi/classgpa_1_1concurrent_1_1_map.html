Class Documentation¶

template<typename KeyType, typename ValueType>
class Map¶

Abstraction for a thread-safe map implementation.

A limited number of operations are currently supported; only what we’ve needed so far. Current implementation is really simple: using a STL unordered_map with a reader-writer lock around the operations.

Public Functions

void Insert(KeyType const &key, ValueType const &value)¶

Insert an element into the map, overwriting any previously set elements for that key.

ReferenceWrapper<ValueType> Get(KeyType const &key)¶

Returns a reference to an existing element in the map.

If no element keys for the key, a default null element is returned instead.
