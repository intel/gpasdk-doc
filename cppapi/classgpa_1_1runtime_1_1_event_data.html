class gpa::runtime::EventData¶

EventData is the base class for all events/messages that can be handled by the EventPublisher dispatch system.

Subclassed by gpa::common::ActionEvent, gpa::common::ApiObjectCreatedEvent, gpa::common::ApiObjectLifetimeEvent, gpa::common::CaptureStreamClosedEvent, gpa::common::CaptureStreamOpenedEvent, gpa::common::ImmediateScreenshotRequest, gpa::common::OutOfDiskSpaceEvent, gpa::runtime::SurfaceCreatedEvent
