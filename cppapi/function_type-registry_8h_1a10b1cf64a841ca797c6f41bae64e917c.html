Template Function gpa::playback::detail::GetTypeId(TypeRegistry<CandidateType, Types…>)¶

  • Defined in File type-registry.h

Function Documentation¶

template<typename T, typename CandidateType, typename ...Types>
constexpr std::enable_if<std::is_same<T, CandidateType>::value, TypeId>::type gpa::playback::detail::GetTypeId(TypeRegistry<CandidateType, Types...> typeRegistry)

Gets the TypeId of a given type from a given TypeRegistry<>

Return

The TypeId of the given type

Note

If the given type is not registered with this TypeRegistry<> the returned TypeId will be 0

Parameters

  • <T>: The type to get the TypeId of

  • [in] typeRegistry: The TypeRegistry<> to get the TypeId of the given type from
